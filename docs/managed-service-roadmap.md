# Future Managed Service Roadmap

This document separates future commercial hosting from the grant-funded open-source toolkit.

Potential future managed service features:

- hosted GasKit instances;
- managed sponsor-wallet operations;
- billing and subscriptions;
- paid support;
- dedicated infrastructure;
- enterprise onboarding;
- SLA-backed monitoring;
- compliance exports.

These features are not required for the grant MVP and should not be represented as grant deliverables.
